The site is at the **repo root**: `index.html`, `css/`, `js/`, `pages/`, `assets/`. Enable GitHub Pages in repo **Settings → Pages** and set source to **main** (root). The site will be at `https://xtraordinaryDev.github.io/S.A.A/`.
